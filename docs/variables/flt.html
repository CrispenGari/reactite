flt is an object that contains filters that can be applied when doing queries and mutations in reactite.

Type declaration

  • between:function
    • between<T>(column, value): {
          stmt: string;
          values: Record<string, number>[];
      }

    • flt.between Filters records based on a range of min and max using the column name specified in the filter.

      Type Parameters

      • T extends number[]

      Parameters

      • column: string

        A string of the column name.

      • value: T

        The value to compare with, which should be an array of exactly 2 elements [min, max].

      Returns {
          stmt: string;
          values: Record<string, number>[];
      }

      • stmt: string
      • values: Record<string, number>[]

      Example

      import { View, Text } from 'react-native'
      import { flt, useQuery } from 'reactite'
      const App = ()=>{
       const { data } = useQuery(
         'users',
         {
           select: ['id', 'avatar'],
           filters: flt.like('id', [2, 10]),
         },
         {
           onData: (data)=>{
       	    console.log({data})
           }
         }
       )

       return (
          <View>
            <Text>Hello Reactite!</Text>
         </View>
       )

Function useDatabaseName

Retrieves the name of the database currently being used by the client. This hook provides the database name which was set during the initialization of the ReactiteClient.

Example

// Usage example within a component
import { useDatabaseName } from "reactite";

const MyComponent = () => {
  const { db } = useDatabaseName();

  return (
    <div>
      <p>Current Database: {db}</p>
    </div>
  );
};

// Renders the name of the database on the screen
